# Go-starter 

## Quick start

```
docker run -p 8020:8020 cloudcarver/wavekit:v0.1.0-pgbundle
```

## Development flow

1. Generate code

```shell
make gen
```

2. Start development environment

```shell
make dev
```

3. Reload code 

```shell
make reload
```

4. Run unit tests

```shell
make ut
```

5. Run end-to-end test

```shell
make test
```
